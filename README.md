# Sin

Sobre: corrige a sincronização de uma legenda de acordo com o tempo em segundos fornecido pelo usuário.

Tecnologia: 
  - C#

Entradas: 
  - Teclado
  - Mouse
  - Arquivos no formato srt.

Plataforma: 
  - Desktop

Requisitos: 
  - Windows 64bits
  - NET Framework
  - Visual Studio Community 2019
